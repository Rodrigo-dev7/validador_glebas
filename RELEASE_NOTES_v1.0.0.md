# Validador de Glebas v1.0.0

Primeira release publica do aplicativo desktop.

## Destaques

- validacao de planilhas `.xls` e `.xlsx`
- deteccao automatica das colunas principais
- identificacao de poligono nao fechado
- identificacao de ponto duplicado em excesso
- identificacao de coordenada invalida
- exportacao de relatorio textual
- executavel `.exe` para Windows

## Arquivos anexados

- `ValidadorGlebas.exe`: executavel principal para uso direto no Windows
- `ValidadorGlebas.zip`: pacote com executavel, exemplos e instrucoes
